# Spectrum VSS Next.js Starter

Editable rebuild of the current Spectrum Veterinary Surgical Services public site.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Edit content

Most page content lives in `lib/site.ts`. Shared layout components are in `components/`.

## Deploy

This is ready for Vercel, Netlify, or any Node hosting provider that supports Next.js.
